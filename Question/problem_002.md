# A - Feet
## AtCoder Beginner Contest 437

### Problem Statement
1 foot is 12 inches.

How many inches is A feet B inches?

### Constraints
- 1 ≤ A ≤ 8
- 0 ≤ B ≤ 11
- All input values are integers.

### Input
The input is given from Standard Input in the following format:
```csharp
A B
```

### Output
Output the answer in one line. Omit the unit (inches).

### Sample Input 1
```csharp
6 7
```

### Sample Output 1
```csharp
79
```
6 feet 7 inches is 6 x 12 + 7 = 79 inches.

### Sample Input 2
```csharp
4 11
```

### Sample Output 2
```csharp
59
```
4 feet 11 inches is 4 x 12 + 11 = 59 inches.

### Sample Input 3
```csharp
8 0
```

### Sample Output 3
```csharp
96
```
8 feet 0 inches is 8 x 12 + 0 = 96 inches.
