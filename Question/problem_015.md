# A - 3,2,1,GO
## AtCoder Beginner Contest 450

### Problem Statement
You are given a positive integer **N**.

Output **N,N-1,...,1** in this order, separated by commas **,**.

### Constraints
- 1 ≤ N ≤ 9
- **N** is an integer.

### Input
The input is given from Standard Input in the following format:
```csharp
N
```

### Output
Output **N,N-1,...,1** in this order, separated by commas.

### Sample Input 1
```csharp
9
```

### Sample Output 1
```csharp
9,8,7,6,5,4,3,2,1
```

### Sample Input 2
```csharp
5
```

### Sample Output 2
```csharp
5,4,3,2,1
```

### Sample Input 3
```csharp
1
```

### Sample Output 3
```csharp
1
```
